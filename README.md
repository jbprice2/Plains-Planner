# The Plains Planner

A personal tracker for Auburn University clubs and events — a lightweight directory of organizations you're following plus a calendar for their events.

## Live app

[https://jbprice2.github.io/Plains-Planner/](https://jbprice2.github.io/Plains-Planner/)

## Features

- **Clubs directory** — save organizations from AuInvolve with category, meeting info, notes, and a link back to their AuInvolve page. Star the ones you're actively following.
- **Calendar** — track event dates, times, and locations, optionally linked to a saved club.
- **On Deck ticker** — a scoreboard-style strip showing your next few upcoming events at a glance.
- **Works offline** — all data is saved locally in your browser, no account or internet connection required after the page loads.

## Adding to your home screen

Open the live link above on your phone:

- **iPhone (Safari)**: Share → Add to Home Screen
- **Android (Chrome)**: Menu (⋮) → Add to Home Screen / Install app

## A note on data

Your clubs and events are stored locally in your browser (`localStorage`). This means:

- Data doesn't sync between devices — each device keeps its own copy.
- Clearing your browser's site data/cache will erase it.

It's a good idea to periodically jot down anything important elsewhere as a backup.

## Editing

The whole app lives in a single file: `index.html`. To make changes, edit that file and re-upload it to this repo — GitHub Pages will update automatically within a minute or two.
